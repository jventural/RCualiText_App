1. QUÉ ES RCualiText
================================================================================
Aplicación web de código abierto en R/Shiny para análisis cualitativo de textos.
Permite cargar documentos (.txt/.docx), codificar fragmentos, crear categorías,
visualizar frecuencias, redes de coocurrencia, y realizar análisis semántico
y codificación asistida por IA (GPT-4.1 de OpenAI).

- URL en producción: https://jventural-rcualitext-app.share.connect.posit.cloud/
- Archivo principal: D:\16_Shinys\RCualiText_App\app.R (~7525 líneas)
- Desplegado en: Posit Connect Cloud (requiere manifest.json)
- Autor: Dr. José Ventura-León

5. SISTEMA i18n (INTERNACIONALIZACIÓN EN/ES)
================================================================================
- reactiveVal("en") como estado global (current_lang)
- Función tr(key, ...) con claves dot-notation: tr("sidebar.documento")
- Interpolación: tr("notifications.codigo_anadido", code = "Alegría")
- ~600 pares clave-valor en ~20 namespaces
- Traducciones embebidas en app.R con r"--(JSON)--" (raw strings R 4.0+)
- Caracteres españoles como Unicode escapes (\u00f3 para ó, etc.)
- Toggle EN/ES: materialSwitch en header (value=FALSE→EN, TRUE→ES)
- Sidebar dinámico: sidebarMenuOutput + renderMenu con tr()
- DataTables: dt_language() traduce search, pagination, etc.
